Google Cloud Setup

  1. Go to console.cloud.google.com → Create a new project named logitrack.
  2. Navigate to APIs & Services → Library → search for Gmail API → Enable.
  3. Go to APIs & Services → OAuth consent screen → External → fill in App Name and your support email → Save.
  4. Go to APIs & Services → Credentials → Create Credentials → OAuth 2.0 Client ID → Application type: Web application.
  5. Under Authorised redirect URIs, add: http://localhost:5001/auth/gmail/callback
  6. Click Download JSON, rename the file to credentials.json, and place it in the project root directory (same folder as app.py).
  7. Generate a Fernet encryption key by running in Python:
    from cryptography.fernet import Fernet; print(Fernet.generate_key().decode())
  8. Set the env var in your shell before starting the server:
    set GMAIL_TOKEN_KEY=<your-key> (Windows CMD)
    or $env:GMAIL_TOKEN_KEY="<your-key>" (PowerShell)
  9. Restart the server, then click Connect Gmail Account above.

AI Processing Setup

Email AI analysis (summarisation, entity extraction, reply generation) uses the Claude API.

  1. Get your API key from console.anthropic.com.
  2. Set the env var:
    set ANTHROPIC_API_KEY=<your-key> (Windows CMD)
    or $env:ANTHROPIC_API_KEY="<your-key>" (PowerShell)
  3. Restart the server. AI features on the email detail page will then be active.

Usage

  • Go to Inbox and click Sync Inbox to fetch your latest Gmail messages.
  • Click any email to view its full content and run AI analysis.
  • Use the Compose Email panel to send emails — vendor contacts are pre-filled in the To field.
  • On the email detail page, click Process with AI to extract shipment references, invoice numbers, and a summary.
  • Click Generate Reply to get a Claude-written reply suggestion.
